# Finite Automata Simulator

A web-based **Finite Automata Simulator** developed using **React, TypeScript, Tailwind CSS, and Konva.js**, designed to provide a rigorous and interactive environment for modeling and analyzing **Deterministic Finite Automata (DFA)** and **Non-Deterministic Finite Automata (NFA)**.

The application combines **theoretical precision** with **production-grade software architecture**, enabling users to visually construct automata, define transitions, and simulate string execution in a stepwise manner.

## Key Features
- **Interactive Canvas-Based Design**: Create and edit DFA/NFA states and transitions through an intuitive visual interface powered by **Konva.js**.
- **Custom Automata Engine**: A strongly typed **TypeScript-based computation engine** implementing formal automata logic with clear separation from the UI layer.
- **Epsilon Transition Support**: Full support for **ε-transitions** in NFA construction and simulation.
- **Step-by-Step Simulation**: Execute input strings incrementally with real-time state traversal and transition highlighting.
- **Formal 5-Tuple Representation**: Live visualization of the automaton as  
  **(Q, Σ, δ, q₀, F)** for theoretical clarity and validation.
- **Trace Table**: Detailed execution trace displaying current state sets, symbols processed, and acceptance status.
- **Preset Automata Examples**: Includes curated examples for learning, testing, and demonstration purposes.

## Academic and Engineering Significance
This project is developed with a strong emphasis on **automata theory pedagogy**, making formal language concepts more accessible through visualization and simulation. Its **decoupled logic–presentation architecture** reflects sound software engineering practices suitable for academic evaluation and scalable extension.
